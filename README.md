# BlogProject
Progetto di test basato su MVC, Repository Pattern, Unit Of Work, HTML5


File di Solution:
BlogProject.sln

Struttura della solution:
	Devevil.Blog.DAL: Data access layer realizzato con i repository concreti attraverso l'ausilio
				di Nhibernate e Fluent Nhibernate
	Devevil.Blog.Infrastructure.Core: Core library contenente classi base e di utilità
	Devevil.Blog.Model: Domain model e business logic
	Devevil.Blog.Unit.Test: Tesi di unità delle librerie

packages:
directory contenente librerie di terze parti referenziate nei projects della solution
